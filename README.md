# -Spotify-Social-Media-Analytics-for-Strategic-Branding-Data-Analysis-MS-Excel
Datasets: Posts Dataset Engagement Summary Dataset Campaign Metadata Dataset 
Project Objective: 
Build a comprehensive Social Media Intelligence Solution in Excel that consolidates Spotify’s 
social data and delivers actionable insights to enhance content strategy, optimize platform 
investments, and evaluate campaign success. 

Dataset Description:
Posts Dataset 
Includes details of social media posts across platforms (e.g., platform, post type, artist tag, likes, 
comments, shares, impressions, hashtags). 
Engagement Summary Dataset 
Weekly aggregated metrics per platform covering ad spend, follower growth, clicks, 
impressions, and ad spends. 
Campaign Metadata Dataset 
Details of Spotify’s major campaigns (e.g.,ChillVibes,Wrapped 2024 etc), including start/end 
dates and objectives.
